What Is Dota 2 Custom Realms?
Dota 2 CR is a lobby client for Dota 2, 100% inspired by the custom games of Warcraft III, allowing anyone to host a custom lobby seamlessly. It is an open source project on Github, so anyone can review the code or contribute to the development of the project.
Getting started
If you want to host and join custom Dota 2 lobbies, you've come to the right place. The first thing you need to do is download our client.
